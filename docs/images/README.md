# Screenshot Capture Guide

## Required Screenshots

Capture the following views for the project README collage:

1. **Configurator overview** — Web configurator main page showing API endpoint configuration.
2. **Discord text generation** — Discord conversation showing the bot responding to a text prompt via Ollama/xAI.
3. **Discord image generation** — Discord conversation showing the bot returning a generated image via ComfyUI or xAI.
4. **Discord tool routing** — Discord conversation showing the bot routing a request to a tool (weather, NFL scores, web search, or meme).
5. **Activity feed** — The `/activity` page showing bot decision events.

## Collage Assembly

- Arrange screenshots in a 2x3 or 2x2 grid.
- Save the final collage as `bot-in-action.png` in this directory.
- Target width: 1200px for the final collage.
- Use a tool like ImageMagick `montage` or any image editor.

## Capture Notes

- Use dark theme in Discord and the configurator for visual consistency.
- Crop to content — remove browser chrome and OS window decorations.
- Target 800-1200px width per individual panel before assembly.
- Ensure no real usernames, API keys, or sensitive data are visible in screenshots.
