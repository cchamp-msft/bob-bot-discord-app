Response Fixup

Some Ollama models return tool calls as XML blocks or bare JSON in their text content instead of using the native tool_calls format. These fixup rules detect and recover those near-miss responses. URL repair fixes common formatting issues like double-wrapped markdown links.

AccuWeather API

Get a free API key at developer.accuweather.com. The AccuWeather API provides current conditions and 10-day forecasts. Configure a default location so users can ask for weather without specifying a city.

NFL API (ESPN)

Powered by ESPN's public API — no API key required. Provides live NFL scores and news. Use tools !nfl scores (optionally with a date like !nfl scores 20260208) or !nfl news (optionally with a filter like !nfl news draft).

Meme API (memegen.link)

Powered by memegen.link — free, no API key required. Templates are cached locally and refreshed every 7 days. Use tool !meme.

SerpAPI (Google Search)

Get a free API key at serpapi.com. Provides Google Search results including AI Overview, Knowledge Graph, and organic results. Use tools !search or !second opinion.

Web Fetch

Fetches web pages and images by URL for the LLM. HTML is parsed to extract text content; direct image URLs are returned as base64. Falls back to SerpAPI search on timeouts, captchas, or blocked responses.

HTTP Server

The Configurator server is the admin interface you are using now (localhost-only by default). The Outputs server is a public-facing HTTP server that serves generated files (images, etc.) so Discord can fetch attachments. Changing any of these values requires a restart.

Security note: this app does not terminate SSL/TLS. Use a trusted reverse SSL proxy for public inbound ports. Do not expose the configurator to the public internet without strict IP filtering and strong admin authentication.
